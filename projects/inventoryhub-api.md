# InventoryHub-API

Inventory management API with clean data models, predictable behavior, and full test coverage.

## What It Does
- Item tracking and categorization
- Stock level management
- Transaction logging
- API endpoints for CRUD operations
- Automated tests for reliability

## Tech Stack
- FastAPI
- SQLAlchemy
- PostgreSQL
- Pytest
- Docker
