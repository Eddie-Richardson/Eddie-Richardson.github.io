# ER-ServiceDesk

Full service desk application with backend and frontend components. Designed for ticket management, role based access, background processing, and reliable data handling.

## What It Does
- Ticket creation, assignment, and tracking
- Role and permission based access
- Background job processing with RQ
- Real time updates through backend events
- Full API layer for integration

## Tech Stack
- FastAPI
- PostgreSQL
- Redis
- RQ
- SQLAlchemy
- Pydantic
- Docker
