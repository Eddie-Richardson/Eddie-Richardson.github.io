# TimeOffCalculator

PTO and PPTO calculation engine used to compute balances, accruals, and rule based time off logic.

## What It Does
- Calculates PTO and PPTO balances
- Applies company specific rules
- Handles edge cases and exceptions
- Provides clean API endpoints for integration

## Tech Stack
- Python
- FastAPI
- Pydantic
- Pytest
